# MenteColetiva

MenteColetiva is a server-side Fabric mod for Minecraft 1.21.11.

## Description

The way Minecraft should be.

Hostile mobs share aggro: when one mob detects or is attacked by a player, nearby hostile mobs can join the attack.

## Features

- shared aggro for nearby hostile mobs
- works even on hitkill
- server-side behavior
- improved skeleton accuracy while the effect is active
- special exclusions for balance:
  - Enderman
  - Zombified Piglin
  - Warden
  - Wither

## Requirements

- Minecraft 1.21.11
- Fabric Loader
- Fabric API

## Environment

Server-side only.
